Exercise 3.

Turn the following vector valued ODE into a system of first order equations.

  1. Given by:

    \begin{equation*} \mathbf{x}^{\prime}=\left(\begin{array}{cc} 3 \amp -2\\ 3 \amp 17 \end{array}\right)\mathbf{x} \end{equation*}

    where \(\boldsymbol{x}(t)=\left(\begin{array}{c} x_{1}(t)\\ x_{2}(t) \end{array}\right)\)

  2. Given by:

    \begin{equation*} \boldsymbol{Y}^{\prime}=\left(\begin{array}{cc} 0 \amp 4\\ 7 \amp 3 \end{array}\right)\boldsymbol{Y} \end{equation*}

    where \(\boldsymbol{Y}(t)=\left(\begin{array}{c} x(t)\\ y(t) \end{array}\right)\text{.}\)
