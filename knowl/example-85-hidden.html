We can now answer the question posed at the beginning of this discussion: since \(e^x = \sum \frac{x^k}{k!}\text{,}\) we get

\begin{equation*} e^{x^2} = \sum \frac{(x^2)^k}{k!} = \sum \frac{x^{2k}}{k!}. \end{equation*}

Then

\begin{align*} \int e^{x^2} \, dx \amp = \int \sum \frac{x^{2k}}{k!}\,dx \\ \amp = \sum \int \frac{x^{2k}}{k!} \, dx \\ \amp = \left[\sum \frac{x^{2k+1}}{(2k+1) k!}\right] + C \end{align*}

This function has no closed form, but the result can be worked with easily (for example, in approximation).
