Suppose that we're given the problem

\begin{equation*} \frac{dy}{dx} = e^{x^2}. \end{equation*}

This is pretty obviously a separable differential equation, and after resorting we get

\begin{equation*} dy = e^{x^2} \,dx, \end{equation*}

and so

\begin{equation*} y = \int e^{x^2} \, dx. \end{equation*}

Unfortunately, the function \(e^{x^2}\) (an enormously useful function that shows up regularly in probability and statistics in the form of the Gaussian or normal distribution), despite being a very nice function, has no closed form antiderivative. That is, there is no function that I can write down for \(y\text{.}\) Can we really not solve a problem like \(\int_0^1 e^{x^2}\, dx\text{?}\)
