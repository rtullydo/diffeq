What are the simplest solutions? An equilibrium solution is a constant solution: \(\mathbf{x}(t)=(x_{0},y_{0})\text{.}\) An equilibrum solution is graphed as dot, since as time moves on, the vector stays constant in the same place. To solve for the equilibrium solution you set the derivatives equal to zero, in this case

\begin{equation*} A\mathbf{x}=\mathbf{0}. \end{equation*}
