In calculus, we learn that one approach to a large class of problems like this is to use power series, which are expressions of the form

\begin{equation*} \sum_{k=0}^\infty a_k(x - a)^k = a_0 + a_1 (x-a) + a_2 (x-a)^2 + \ldots \end{equation*}

where \(a_0, a_1,\ldots\) are constants. (We'd really like to think of a power series like giant polynomial, but this is isn't always justified...)
