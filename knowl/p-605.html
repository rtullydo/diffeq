So how do we find them? From the geometry of the phase plane. If \(\mathbf{x}(t)\) is a straight-line solution, then notice that \(A\mathbf{x}=\lambda\mathbf{x}\) for some \(\lambda\text{.}\) Because the vector \(A\mathbf{x}\) points in the same direction as the vector from \((0,0)\) to \(\mathbf{x}\text{.}\) This means the solution \(\mathbf{x}(t)\) are formed from eigenvectors. So if we can find an eigenvector and its eigenvalue then we would have found a straight line solution.
