Everything we did in ChapterÂ 3for second order linear equations can be extended to higher order systems. Suppose we have the \(n\)th order linear equation

\begin{equation*} a_{n}(t)y^{(n)}+a_{n-1}(t)y^{(n-1)}+\cdots+a_{1}(t)y^{\prime}+a_{0}(t)y=g(t). \end{equation*}

We assume that \(a_{n}(t),\dots,a_{0}(t)\) are continuous functions on an interval \(I\text{,}\) and that \(a_{n}(t)\neq0\) inside the interval: so that we can write it in standand form as

\begin{equation*} y^{(n)}+p_{n-1}(t)y^{(n-1)}+\cdots+p_{1}(t)y^{\prime}+p_{0}(t)y=g(t).\,\,\,\,\,(\star) \end{equation*}

with initial conditions

\begin{equation*} y(t_{0})=y_{0},\,\,y^{\prime}(t_{0})=y_{0}^{\prime},\,\,\cdots,y^{(n-1)}(t_{0})=y_{0}^{(n-1)}.\,\,\,\,\,(\star)\text{.} \end{equation*}
