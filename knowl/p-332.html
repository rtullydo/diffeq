Using differential operator notation allows us to transform an ODE and use algebraic techniques to solve it using first order methods. Consider the equation

\begin{equation*} y'' + 3 y ' + 2 y = 0. \end{equation*}

This can be written

\begin{equation*} D^2 y + 3 D y + 2 y = 0 \end{equation*}

using operator notation. But since each operator is applied to the same input, \(y\text{,}\) we can use function notation to write

\begin{equation*} (D^2 + 3D + 2)(y) = 0. \end{equation*}
