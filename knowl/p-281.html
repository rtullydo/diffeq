In summary, given a potential function \(z = f(x,y)\text{,}\)

  1. We can find the equipotential lines (lines of constant height)
    \begin{equation*} C = f(x,y); \end{equation*}
  2. we can find the gradient field
    \begin{equation*} \nabla f = \langle f_x, f_y \rangle; \end{equation*}
  3. and we know that at a given point, the equipotential and the gradient line are perpendicular.
