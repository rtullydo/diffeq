The pictures above are a derivation of the differential equation for a simple mass-spring system. Notice that the resulting equation has a second derivative \(x''\) in it - thus, this is a second order equation where the position of the mass \(x(t)\) relative to the equilibrium position is a function of time \(t\text{:}\)

\begin{equation*} x'' + \frac{k}{m} x = 0. \end{equation*}

Also, note the important fact that the equation is linear and has constant coefficients. If we want to describe how the solutions to this equation behave, we should study second order equations with constant coefficients.
