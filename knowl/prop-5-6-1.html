Proposition 5.6.3.

If a linear system \(\mathbf{x}^{\prime}=A\mathbf{x}\) has an eigenvalue \(\lambda=\alpha\pm\beta i\text{.}\) Then the solution curves form spirals about the origin with natural period \(\frac{2\pi}{\beta}\text{,}\) natural frequency \(\frac{\beta}{2\pi}\) and

(1) If \(\alpha \lt 0 \text{,}\) we call this an asymptotically stable spiral point. All solutions spiral in towards the origin \(\mathbf{0}\text{.}\)

(2) If \(\alpha \gt 0\text{,}\) we call this an asymptotically unstable spiral point. All solutions spiral away from the origin \(\mathbf{0}\text{.}\)

(3) If \(\alpha = 0\text{,}\) we call this a center. All solutions are elliptical shaped.
