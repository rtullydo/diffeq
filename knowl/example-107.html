Example 5.4.7.

Find the Laplace transform of

\begin{equation*} f(t)=\begin{cases} 0 \amp t\lt 2\\ t^{2}-4t+5 \amp t\geq2 \end{cases} \end{equation*}

Solution: First we complete the square by adding/subtracting \(\left(\frac{b}{2}\right)^{2}=\left(\frac{4}{2}\right)^{2}=4\) and get

\begin{equation*} t^{2}-4t+5=t^{2}-4t+4-4+5=\left(t-2\right)^{2}-4+5=(t-2)^{2}+1 \end{equation*}

so that

\begin{align*} f(t) \amp =\begin{cases} 0 \amp t\lt 2\\ \left(t-2\right)^{2}+1 \amp t\geq2 \end{cases}\\ \amp =u_{2}(t)\left[\left(t-2\right)^{2}+1\right]\\ \amp =u_{2}(t)\left(t-2\right)^{2}+u_{2}(t), \end{align*}

hence using formulas \(\mathcal{L}\left\{ u_{c}(t)f\left(t-c\right)\right\} =e^{-cs}F(s)\) and \(\mathcal{L}\left\{ t^{2}\right\} =\frac{2}{s^{3}}\) and \(c=2\text{,}\)

\begin{align*} \mathcal{L}\left\{ f(t)\right\} \amp =e^{-2s}F(s)+\frac{e^{-2s}}{s},\,\,\text{where }f(t-2)=\left(t-2\right)^{2},f(t)=t^{2}\\ \amp =e^{-2s}\frac{2}{s^{3}}+\frac{e^{-2s}}{s}. \end{align*}
