The technique we'll use in this section will only work if the first order ODE is separable. We say a first order ODE is separable if we can write it in the following form:

\begin{equation} \frac{dy}{dt}=g(t)h(y).\tag{2.2.1} \end{equation}

If we can write it this way, then separate the variables to get (put all the \(y\)'s on one side and all \(t\)'s on the other side)

\begin{equation*} \frac{1}{h(y)}dy=g(t)dt, \end{equation*}

and then integrate both side with respect to their respective variable. This is legal by a \(u\)-substitution argument. (This is informal algebra!)
